# Cool Blog 🌟

一个现代、酷炫的博客网站，用代码创造世界！

## 特性 ✨

- 🎨 **现代设计** - 深色模式、渐变色彩、流畅动画
- 🌙 **主题切换** - 支持深色/浅色主题
- 📱 **响应式布局** - 完美适配各种设备
- ⚡ **高性能** - 纯原生 HTML/CSS/JS，无依赖
- 💫 **交互动画** - 滚动动画、悬停效果、平滑过渡

## 技术栈 🛠️

- **HTML5** - 语义化标签
- **CSS3** - CSS 变量、Flexbox、Grid、动画
- **JavaScript** - 原生 JS，无框架

## 快速开始 🚀

1. 克隆这个仓库
2. 直接在浏览器中打开 `index.html`
3. 开始探索！

## 项目结构 📁

```
my_cool_blog/
├── index.html          # 主页
├── styles.css          # 样式文件
├── script.js           # 交互逻辑
├── posts/              # 文章文件夹
├── assets/             # 资源文件夹
└── README.md           # 项目说明
```

## 功能预览 🎯

- 🏠 **首页英雄区** - 震撼的欢迎界面
- 📝 **文章网格** - 展示最新博客文章
- 👤 **关于页面** - 了解更多关于这个博客
- 🎯 **导航系统** - 固定顶部导航，支持锚点跳转

## 自定义 ✏️

- 在 `script.js` 中修改 `posts` 数组来添加你的文章
- 在 `styles.css` 中调整 CSS 变量来改变配色方案
- 在 `index.html` 中编辑内容来个性化你的博客
